# learn-clojure

## Clojure Basics

This project contains code and documentation for a introductory
Clojure course.

The following sections are described:

  * The REPL
  * Clojure syntax and the function call
  * Boolean values and `nil`
    * Logical truth
  * Numbers
    * Integer values
    * Decimals values
      * Floating point
      * BigDecimal
    * Rationals values
  * Characters
  * Strings
  * Keywords
  * Collections
    * Lists
    * Vectors
    * Maps
    * Sets
  * Regular expression patterns
  * Symbols
  * Immutability
  * Functions
    * what is a function
    * arity
    * scope
    * high-order functions
    * anonymous functions
    * lambdas
    * recursion
  * Vars, scope, and local bindings
  * The sequence abstraction
  * Destructuring
  * Flow control
    * if
    * when
    * cond
    * case
  * Core functions
    * map
    * apply
    * reduce
    * filter
    * group-by
    * sort
    * partition
    * into
  * Operations with files
    * slurp
    * spits
